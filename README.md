> Slice an image to frames and pixels

## Installation
```bash
$ npm install pixel
```

## Installation
### Via npm
```bash
$ npm install pixel --save
```
```js
var Pixel= require('pixel');
console.log(Pixel); //function
```

### Via bower
```bash
$ bower install pixel --save
```
```html
<script src="bower_components/pixel/pixel.min.js"></script>
<script>
  console.log(Pixel); //function
</script>
```
